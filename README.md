⚡ Campus Complaint Tracker

🏫 Project Description

Campus Complaint Tracker is a simple smart contract built on Stellar Soroban, designed to provide a transparent and immutable reporting system for campus-related issues. Students can submit reports regarding facilities or services, and each report is permanently stored on the blockchain with a trackable status. This ensures that every complaint is not only recorded but can also be openly audited without the risk of data manipulation.

🔭 Project Vision
"Transparency in every report."
We envision a more open and responsive campus environment where every student’s voice is recorded and can be clearly monitored. The goals of this project are:

- Empowering Student Voices → Providing a transparent platform for students to submit complaints
- Structured Reporting → Converting reports into structured and trackable data
- Immutable Transparency → Ensuring reports cannot be altered or deleted unilaterally
- Accountability System → Encouraging campus authorities to respond more effectively

🛠 Key Features
- [REPORT CREATION] → Students can create reports with a title and description
- [STATUS TRACKING] → Each report has a status: pending, in progress, completed
- [ON-CHAIN STORAGE] → Data is permanently stored on the blockchain
- [TRANSPARENT ACCESS] → All reports can be accessed and monitored
