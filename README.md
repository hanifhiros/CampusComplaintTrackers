⚡ PROJECT TITLE: CAMPUS COMPLAINT TRACKER

🏫 PROJECT DESCRIPTION
Campus Complaint Tracker adalah smart contract sederhana berbasis Stellar Soroban yang berfungsi sebagai sistem pelaporan permasalahan kampus yang transparan dan immutable. Mahasiswa dapat membuat laporan terkait fasilitas atau layanan kampus, sementara setiap laporan disimpan secara permanen di blockchain dengan status yang dapat dipantau. Dengan pendekatan ini, setiap keluhan tidak hanya tercatat, tetapi juga dapat diaudit secara terbuka tanpa risiko manipulasi data.

🔭 PROJECT VISION
"Transparency in every report."

Kami membayangkan lingkungan kampus yang lebih terbuka dan responsif, di mana setiap suara mahasiswa tercatat dan dapat dipantau secara jelas. Tujuan dari project ini adalah:

Empowering Student Voices: Memberikan ruang bagi mahasiswa untuk menyampaikan keluhan secara transparan
Structured Reporting: Mengubah laporan menjadi data terstruktur yang mudah dilacak
Immutable Transparency: Menjamin laporan tidak dapat diubah atau dihapus secara sepihak
Accountability System: Mendorong pihak kampus untuk lebih responsif terhadap laporan

🛠 KEY FEATURES

[REPORT CREATION] - Mahasiswa dapat membuat laporan dengan judul dan deskripsi
[STATUS TRACKING] - Setiap laporan memiliki status: pending, proses, selesai
[ON-CHAIN STORAGE] - Data tersimpan permanen di blockchain
[TRANSPARENT ACCESS] - Semua laporan dapat diakses dan dipantau
